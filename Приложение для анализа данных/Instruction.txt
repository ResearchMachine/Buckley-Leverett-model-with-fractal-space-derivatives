Инструкция по использованию:
Для отображения вкладок моделирования и визуализации необходимо загрузить файл с данными
----------------------------
1. Загрузка данных (.xlsx)
а) Подготовьте файл с данными xlsx: вторая строка обязательно должна быть указана в верном формате 
(см. пример в корневой папке прототипа)
б) Укажите диапазоны загружаемых строк и столбцов, символы пустой ячейки и количество строк для вывода
в) Нажмите Загрузить данные
Внимание! Перед дальнейшей работой проверьте правильность определения форматов переменных, иначе могут возникнуть ошибки
----------------------------

----------------------------
2. Моделирование
2.1 Корреляционный анализ
а) Выберите коэффициент корреляции
(в построении матрицы используются только числовые переменные)
б) Нажмите Выполнить

2.2 Иерархический кластерный анализ
а) Выберите метод расчета расстояния между кластерам
б) Выберите переменные по которым производится кластеризация
(выбор только среди числовых переменных)
в) Нажмите Выполнить
Вывод: 
i) Дендрограмма 
Дерево, построенное по матрице мер близости между кластерами. 
В узлах дерева находятся подмножества объектов из обучающей выборки. 
При этом на каждом ярусе дерева множество объектов из всех узлов составляет исходное множество объектов. 
Объединение узлов между ярусами соответствует слиянию двух кластеров. 
При этом высота ребра соответствует расстоянию между кластерами.
ii) Показатель силуэтта 
Значение силуэта показывает, насколько объект похож на свой кластер по сравнению с другими кластерами.

2.3 Дискриминантный анализ
а) Выберите зависимую переменную 
(выбор только среди категориальных переменных)
б) Выберите независимые переменные 
(выбор только среди числовых переменных)
в) Нажмите Выполнить

2.4 Линейная многомерная регрессионная модель
(в построении модели используются только числовые переменные)
а) Выберите зависимую переменную 
б) Выберите независимые переменные
в) Выберите метод заполнения пустых ячеек в независимых переменных
г) Строки с пустыми ячейками в зависимых переменных не учитываются
д) Введите величину коэффициента корреляции по коэффциенту Спирмена по модулю
(теснота линейной связи, 1 - точная линейная связь, 0 - полное отсутвие связи)
е) Введите величину значимости по p-value (*)
ж) Введите величину значимость по VIF критерию (*)
з) Нажмите Выполнить
(*) Внимание! В данных ячейках по умолчанию установлены рекомендуемые параметры
Внесение в них изменений может привести к ошибочным результатам моделирования!
Рекомендуется обратиться к ML-специалисту.

2.5 Сэмплирование.
а) Укажите диапазон варьирования параметров модели(**) 
б) Укажите закон распределния и количество генерируемых наблюдений
По умолчанию установлены границы +-10% от средних значений внутри текущей
базы данных
